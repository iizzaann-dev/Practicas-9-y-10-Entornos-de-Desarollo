## 📌 About This Project 💻

This project is part of my programming coursework. It includes:

- 🧮 A **Calculator class** with basic arithmetic operations
- 📐 A **Rectangle class** to calculate area (surface) and perimeter
- 🧪 **JUnit 5 tests** for unit testing
- 📊 **EclEmma coverage analysis**
- 🧼 **Refactoring practices**
- 🧠 **SonarLint code quality analysis**
- 📚 **Javadoc documentation**

The goal of this project is to learn how to write **clean, tested and maintainable code** in Java.
